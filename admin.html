System Setup Guide

Phase 1: Physical Setup & First Login

Hardware: Internet In to Port 1. Computer to Port 2.
Login: Use WinBox, click Neighbors tab, login: admin / no password.
Reset: Click Remove Configuration upon first login.

Phase 2: Connecting to the Internet

WAN: IP > DHCP Client > [+], select ether1.
Routes: IP > Routes > [+], set Gateway IP.
DNS: IP > DNS, add 8.8.8.8 and 8.8.4.4. Allow Remote Requests.

Phase 3: Creating the Hotspot Network

Bridge: Bridge > [+] name: bridge1. Add ether2 and wlan1 to ports.
Wizard: IP > Hotspot > Hotspot Setup. Select bridge1. Set DNS Name: snookum.wifi.

Phase 4: Linking the Billing System

Enable API: IP > Services > Enable api (8728).
Terminal: Paste the Script from the MikroTik Setup tab into a New Terminal.
Radius: In Hotspot Profile, Login tab: check PAP only. Radius tab: check Use Radius.

Phase 5: Uploading Portal Files

Files: Open Files in WinBox. Find hotspot folder.
Backup: Rename original login.html to login_backup.html.
Upload: Drag your custom login.html into the hotspot folder.

Verification: Connect phone to Wi-Fi. Portal should pop up automatically. If not, check DNS settings.
